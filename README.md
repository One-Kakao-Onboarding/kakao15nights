# UX-Ray 👁️

> AI 페르소나가 당신의 UI를 진단하고, 놓치기 쉬운 접근성 문제를 시각적으로 보여드립니다.

## 💡 프로젝트 소개

UX-Ray는 사용자 테스트 없이도 다양한 사용자 관점에서 UI/UX 문제점을 발견할 수 있는 AI 기반 진단 솔루션입니다.
4가지 AI 페르소나가 UI 스크린샷을 분석하고, **Visual Red Pen** 방식으로 문제 영역을 직관적으로 표시합니다.

### 왜 UX-Ray인가요?

- 🎯 **실제 사용자 관점**: 고령층, ADHD, 한손 사용자, 외국인 등 다양한 페르소나의 시각
- ⚡ **빠른 피드백**: 평균 30초 내 분석 완료
- 📊 **시각화된 결과**: 문제점을 좌표와 함께 이미지 위에 직접 표시
- 🌍 **디지털 포용**: 1,200만 명의 디지털 소외 계층을 위한 접근성 개선

## 🛠️ 기술 스택

React + TypeScript 기반의 웹 애플리케이션이며, Google Gemini AI를 활용하여 UI 이미지를 분석합니다.

## 📸 사용 방법

1. **UI 스크린샷 업로드** - 웹/앱 화면을 이미지로 업로드
2. **페르소나 선택** - 진단받고 싶은 AI 페르소나 선택 (다중 선택 가능)
3. **결과 확인** - 각 페르소나의 피드백과 문제점 좌표를 시각적으로 확인

## 🌟 주요 기능

- ✅ **멀티 페르소나 분석**: 최대 4명의 페르소나 동시 진단
- ✅ **Visual Red Pen**: 문제 영역을 이미지 위에 직접 표시
- ✅ **고유 능력 시뮬레이터**: 각 페르소나별 특수 시각화 기능
- ✅ **PDF 리포트 생성**: 분석 결과를 PDF로 저장
- ✅ **데모 이미지 제공**: 빠른 체험을 위한 샘플 이미지
